# 給心理工作者的 AI 入門課：在自己的電腦上，累積你的教案資料庫

> 這份講義是給「你」看的——一位想用 AI 幫忙，但還沒碰過程式、第一次聽到 Antigravity 的心理工作者。
> 全程用白話，不需要任何程式基礎。跟著步驟做，你會親手完成你的第一個「AI 看得懂的教案資料庫」。
> 使用工具：Gemini Antigravity（這堂課的主角，一個能讀你電腦檔案的 AI 工具）

---

## 這堂課結束後，你會做到三件事

1. 看懂「為什麼要把 AI 從網頁，搬到自己的電腦上」。
2. 在自己電腦建好一個整理過的資料夾，讓 AI 能讀、能幫你分析。
3. 用 AI 整理出第一份統一格式的教案，並產出一份投影片大綱。

> 這堂課「不做」的事：不碰個案紀錄、不寫程式、不追求複雜系統。我們只專心打好地基——累積資料、做教案、做投影片。

## 單元 1：為什麼要把 AI 搬到自己的電腦上？

### 1.1　你現在大概是這樣用 AI 的

打開瀏覽器 → 進 ChatGPT 或 Gemini 網頁 → 貼上內容、問問題 → 得到回答 → 關掉視窗。

這樣很方便，但你可能也遇過這些狀況：

- 上週請它幫忙改的教案，這週它完全不記得了，又要重講一遍。
- 三個月前它給過一個很棒的活動點子，現在翻對話紀錄怎麼也找不到。
- 它只能「用講的」給你建議，沒辦法真的幫你把一疊資料整理好、存起來。

### 1.2　一個簡單的比喻

| | 網頁版 AI | 在你電腦上的 AI |
|---|---|---|
| 它像誰 | 一位很聰明、但每次見面都失憶的路人顧問 | 一位坐在你辦公室、看過你所有檔案的助理 |
| 它看得到什麼 | 只有你「這次」貼給它的那一段 | 你整個資料夾裡累積的教案與筆記 |
| 講完之後 | 對話關掉就忘了 | 成果存成檔案，永遠留在你電腦裡 |

### 1.3　一句話講重點

> 網頁版 AI 每次都從零開始。
> 在你電腦上的 AI，會站在你「過去累積」的基礎上，幫你做得更快、更貼近你的風格。

**這堂課要做的，就是幫你把「過去的累積」放到一個 AI 看得到的地方。**

## 單元 2：核心觀念——什麼是「資料叢集」？

### 2.1　先看你現在的東西放在哪

很多人的教案與素材，現在大概是這樣散落各處：

```
你的教案與素材，現在散落在……

  📧 信箱附件裡的某份 Word
  💬 跟同事的對話紀錄
  📁 桌面上「新增資料夾(3)」
  🖥️ 隨身碟某個角落
  ☁️ Google 雲端硬碟還沒整理的那一堆
  🧠 你的腦袋（最不可靠的地方）

  → AI 看不到這些，你自己也常常找不到。
```

### 2.2　「資料叢集」就是把它們收成一團、放整齊

「資料叢集」聽起來很專業，其實就是一句話：**一個整理好、分好類的資料夾**。

```
散落各處              整理成資料叢集
（找不到、AI 讀不到）   （好找、AI 一次讀得到）

  📄 📄                ┌─────────────────┐
   📄    📄            │ 📁 我的教案庫      │
 📄   📄      ──→      │  ├ 📁 情緒主題     │
    📄  📄             │  ├ 📁 親子主題     │
  📄      📄           │  └ 📁 投影片素材   │
                       └─────────────────┘
```

### 2.3　為什麼「整理過」對 AI 特別重要

當你的資料有清楚的資料夾名稱、檔名、分類，AI 讀的時候等於**先拿到一張地圖**，它知道哪裡放什麼，整理和比對就又快又準。

舉個例子，當你問 AI：

> 「我過去做過的情緒主題教案，有哪些暖身活動？」

- 資料還在散落狀態：AI 看不到，你得自己一份份翻、一份份貼給它。
- 已經收成資料叢集：AI 直接讀完「情緒主題」資料夾裡所有教案，幫你整理成一張清單。

**結論：你越花一點心思分類，AI 就越幫得上忙。這是雙向的投資。**

## 單元 3：第一次設定（給完全新手的步驟）

> 別緊張。這一段你只要會「點滑鼠」和「打字」就能完成。
> 提醒：不同版本的畫面文字可能略有差異，看到類似意思的按鈕就對了。

### 步驟 0：安裝 Antigravity

1. 用瀏覽器搜尋「Antigravity」，進入官方下載頁。
2. 下載對應你電腦的版本（Windows 或 Mac），照畫面指示安裝。
3. 安裝完成後，打開它。第一次打開可能要用 Google 帳號登入。

### 步驟 1：先在電腦上建一個「家」（資料夾）

在開始用 AI 之前，先準備好它要讀的資料夾。

1. 在你的「文件」或「桌面」按右鍵 → 新增資料夾。
2. 取一個好認的名字，例如 `我的教案庫`。
3. 在裡面，先建幾個分類子資料夾，例如：

```
我的教案庫\
├── 情緒主題\
├── 親子主題\
├── 投影片素材\
└── 草稿與點子\
```

> 不用一次想到完美。先建 2 至 3 個你最常做的主題就好，之後隨時能加。

### 步驟 2：用 Antigravity 打開這個資料夾

Antigravity 的畫面，大致長這樣：

```
┌──────────────┬─────────────────────────────┐
│              │                             │
│  左邊：       │   右邊：                     │
│  你的資料夾    │   跟 AI 對話的地方            │
│  與檔案清單    │   （在這裡打字交代它做事）     │
│              │                             │
│  我的教案庫    │   你：幫我把這份教案整理成…    │
│  ├ 情緒主題   │   AI：好的，我看到了…         │
│  ├ 親子主題   │                             │
│              │                             │
└──────────────┴─────────────────────────────┘
```

操作：

1. 在 Antigravity 上方選單找到「打開資料夾 / Open Folder」。
2. 選你剛剛建立的 `我的教案庫`。
3. 左邊就會出現你的資料夾與檔案——這代表 AI 現在「看得到」這些東西了。

### 步驟 3：第一次跟它說話

在右邊的對話框，試著打第一句話，確認它真的讀到了你的資料夾：

> 「請看一下這個資料夾，告訴我裡面現在有哪些檔案和子資料夾。」

如果它能正確說出你建的那幾個子資料夾，恭喜——設定完成，地基打好了。

## 單元 4：第一個成果——整理你的第一份教案

這一單元，我們把「一份亂亂的舊教案」變成「乾淨、統一格式的教案」。

### 步驟 1：把舊教案放進資料夾

把你手邊任何一份現成教案（Word、PDF、甚至只是純文字筆記）複製到對應的子資料夾，例如 `情緒主題\`。

### 步驟 2：請 AI 整理成統一格式

在對話框輸入類似這樣的話（直接複製去改）：

> 「請讀 `情緒主題` 資料夾裡的那份教案，幫我整理成統一格式，包含這幾個欄位：主題、對象、時間、目標、暖身活動、主要活動、討論問題、結束與回家作業。整理好存成一個新的 Markdown 檔。」

### 步驟 3：看看前後差別

```
整理前（你丟進去的）          整理後（AI 幫你產出的）
──────────────────          ──────────────────────
一大段文字，               # 情緒辨識教案
活動跟目標混在一起，         - 對象：國中生
時間沒寫清楚，      ──→     - 時間：90 分鐘
格式每份都不一樣。           - 目標：能說出三種情緒
                          ## 暖身活動（15 分鐘）
                          ……（欄位清楚、好讀好找）
```

### 步驟 4：建立你的「教案模板」（重要的一步）

請 AI 把剛剛的格式，另存成一份空白模板，以後每份教案都照它做，格式就會統一：

> 「把剛剛的格式，做成一份空白的教案模板，存成 `_教案模板.md`，欄位留空讓我之後填。」

> 小技巧：檔名前面加一個底線 `_`，它通常會排在資料夾最上面，好找。

**做到這裡，你已經完成兩件事：第一份整齊教案 ＋ 一個可重複使用的模板。**

## 單元 5：從累積內容，到一份完整投影片

這一單元最重要，分成「兩條路」：

- **主流程（5.1 至 5.4）：** 大綱 → 你確認修改 → 指定風格配色生成 → 你再微調。
- **進階（5.5）：** 把你過去的投影片，反向做成「風格模板規則」，讓以後生成的投影片都長得一致。

```
主流程：
 教案 → 產大綱 → 你確認/改 → 指定風格配色生成 → 你微調 → 完成

進階（讓風格統一）：
 過去的投影片 → 請 AI 分析 → 產出「風格規則」→ 之後生成都遵守
```

> 你永遠是主導者：AI 出草稿，你做決定。下面每一步都附「可直接複製去改的指令」。

### 5.1　第一步：請 AI 依教案產出投影片大綱

> 「依照剛剛整理好的這份情緒辨識教案，幫我規劃一份投影片大綱。每一張投影片給我：標題、3 至 5 個重點、以及這張要帶的活動或提問。」

你會拿到像這樣的骨架：

```
第 1 張｜課程介紹
  - 今天的主題：認識自己的情緒
  - 三個目標、一句開場提問

第 2 張｜暖身：情緒猜猜看
  - 規則說明、帶一個例子、預留 10 分鐘
```

### 5.2　第二步：確認並修改大綱（這步別跳過）

AI 給的是草稿，**你一定要讀過、改成你要的**。直接用白話告訴它哪裡要動，一次講清楚：

> 「這份大綱方向我喜歡。但有兩個地方要改：第 3 張的活動換成『角色扮演』；第 5 張內容太多，幫我拆成兩張。其他不變，更新給我。」

> 「第 2 張的重點太多，精簡成 3 點，而且用更口語、適合對國中生說的語氣。」

改到你滿意，再進下一步。

### 5.3　第三步：指定細節、風格、配色，請 AI 生成投影片

大綱定案後，把你要的「樣子」講得越具體，成品越接近你想要的。一個好的生成指令，通常包含這幾塊：**對象、語氣、配色、每頁規則、要不要講者備註、輸出成什麼**。

範例（複製後改成你的需求）：

> 「請依照這份已確認的大綱，幫我做出完整的投影片內容。要求：
> 1. 對象與語氣：國中生，溫暖、輕鬆，不要太正式。
> 2. 配色：主色用大地綠，輔色用暖橘。
> 3. 每張投影片：一個主標題、最多 4 個短句重點（每句不超過 15 字）、底部一句引導提問。
> 4. 每張後面附『講者備註』，寫我上課時可以怎麼說。
> 5. 做成一份我可以預覽的投影片草稿檔。」

> 老實說：AI 能把「每一頁的內容、順序、風格規則」做到八成並產出草稿。但最後的精細排版、動畫、換圖，通常還是在 PowerPoint 或 Google 簡報裡完成。把 AI 當成「幫你把骨架和血肉都長好」的人，最後一哩的化妝由你來。

### 5.4　第四步：拿到草稿後，自己再微調

不滿意？繼續用白話請它改，一次改一個地方最好掌握：

> 「整體不錯，但語氣再活潑一點，每張幫我加一個適合的 emoji 當視覺提示。」

> 「第 4 張太滿了，幫我拆成兩張，節奏放慢。」

> 「所有投影片的頁尾，統一加上機構名稱和頁碼。」

### 5.5　進階：用過去的投影片，做出你的「風格模板規則」

如果你手邊已經有做過的投影片，可以反過來——**讓 AI 學你的風格**，以後生成的都自動統一。

**步驟 A：把舊投影片放進素材夾。** 把幾份過去的投影片（或匯出的 PDF）放進 `投影片素材\`。

**步驟 B：請 AI 分析並寫成規則。**

> 「我把過去幾份投影片放進了『投影片素材』資料夾。請閱讀它們，分析我的排版與風格習慣：常用的標題大小、配色、每頁的文字量、慣用的版面結構。整理成一份『投影片風格規則』，存成 `_投影片風格規則.md`。」

**步驟 C：以後生成都套用這份規則。**

> 「以後我請你做投影片時，一律遵守 `_投影片風格規則.md` 裡的規則。」

> 「請依照 `_投影片風格規則.md` 的規範，把這份新教案做成投影片。」

這樣一來，不管哪一天、做哪個主題，你的投影片都會有一致的樣子——這就是你的「風格指紋」。

### 5.6　別忘了：素材也收進資料夾

把常用的圖片、案例、講義丟進 `投影片素材\`。下次請 AI 規劃時，它就能建議「這張可以放你之前那張情緒溫度計圖」。

## 單元 6：養成習慣，讓你的資料庫越用越好用

這套方法的威力，不在第一天，而在「持續累積」。重點是養成一個小習慣：

```
        每次做完一件事
              │
              ▼
   把成果存回「我的教案庫」對的資料夾
              │
              ▼
     下次 AI 就多讀到一份你的累積
              │
              ▼
        它的建議越來越像「你」
              │
              └──────────┐
                         │（不斷循環）
              ┌──────────┘
              ▼
        資料庫越長越大，越好用
```

### 一個簡單的每週小流程

**平常（隨手做）：** 任何教案、點子、好用的活動，隨手存進對應資料夾。

**每週一次，請 AI 幫你看一遍：**

> 「幫我看一下這個資料夾，這週新增了哪些檔案？有沒有放錯位置、或還沒分類好的？給我建議。」

**每月一次，請 AI 做跨教案整理：**

> 「把我所有情緒主題教案裡的『暖身活動』，整理成一張清單，並標出哪幾個重複、哪幾個我可能可以合併。」

### 累積三個月後，你會感覺到的變化

| 時間 | 你的資料庫 | AI 幫你的程度 |
|---|---|---|
| 第 1 週 | 幾份教案 ＋ 一個模板 | 幫你整理格式 |
| 第 1 個月 | 十幾份教案、分好類 | 開始能引用你過去的內容 |
| 第 3 個月 | 各主題都有累積 | 新教案做得又快又像你的風格 |

## 附錄 A：下一個「好指令」的小公式

一開始不知道怎麼跟 AI 講？記住這個公式，把四塊填滿就行：

```
【要它做什麼】＋【根據哪份資料／資料夾】＋【要符合什麼條件或格式】＋【結果存去哪】
```

對照範例：

| 公式的一塊 | 你可以怎麼寫 |
|---|---|
| 要它做什麼 | 「幫我整理成統一格式」 |
| 根據哪份資料 | 「情緒主題資料夾裡那份教案」 |
| 要什麼條件／格式 | 「包含目標、暖身、主要活動、討論」 |
| 結果存去哪 | 「存成一個新的 Markdown 檔」 |

把四塊接起來，就是一個清楚的好指令。**越具體，AI 越不會亂猜。**

三個小訣竅：

- 一次只交代一件事，做完再下一個。
- 不滿意就直接說哪裡不對，請它改——它不會生氣，可以一直改。
- 看不懂它做了什麼，就問：「你剛剛做了什麼？用白話跟我說。」

## 附錄 B：提示詞參考卡（可直接複製）

把這張當小抄。需要時複製，把〔括號〕內換成你的內容。

**A. 開始與確認**

> 「請看一下這個資料夾，告訴我裡面有哪些檔案和子資料夾。」

**B. 整理教案**

> 「請讀〔某資料夾〕裡那份教案，整理成統一格式，包含：主題、對象、時間、目標、暖身活動、主要活動、討論問題、結束與回家作業。存成新的 Markdown 檔。」

> 「把剛剛的格式做成空白模板，存成 `_教案模板.md`，欄位留空讓我之後填。」

**C. 投影片：生成大綱**

> 「依照〔某份教案〕，規劃投影片大綱。每張給我：標題、3 至 5 個重點、要帶的活動或提問。」

**D. 投影片：修改大綱**

> 「方向我喜歡，但〔第幾張〕改成〔……〕，〔第幾張〕拆成兩張，其他不變，更新給我。」

**E. 投影片：指定風格生成**

> 「依照已確認的大綱做出完整投影片內容。對象〔……〕、語氣〔……〕、配色〔主色／輔色〕、每張最多 4 個短句重點、附講者備註、做成可預覽的草稿檔。」

**F. 投影片：拿到草稿後微調**

> 「語氣再活潑一點，每張加一個合適的 emoji。」

> 「〔第幾張〕太滿，拆成兩張。」

> 「所有頁尾統一加機構名稱和頁碼。」

**G. 投影片：讓 AI 學你的風格、做出規則**

> 「閱讀『投影片素材』裡我過去的投影片，分析我的標題大小、配色、每頁文字量、版面結構，整理成 `_投影片風格規則.md`。」

> 「以後做投影片一律遵守 `_投影片風格規則.md`。」

**H. 持續累積與整理**

> 「看一下這個資料夾，這週新增了什麼？有沒有放錯或沒分類好的？」

> 「把所有〔某主題〕教案的〔某個環節〕整理成一張清單，標出重複的。」

## 常見疑問

**Q1：我完全不會寫程式，真的可以嗎？**
可以。你從頭到尾都是「用中文打字交代它做事」，不需要寫任何程式碼。

**Q2：我的個案紀錄可以丟進去嗎？**
這堂課請「先不要」。個案資料牽涉保密與倫理，需要更謹慎的設定（離線、去識別化），那是進階主題。**先用沒有敏感資料的教案與投影片練熟，把工具用順了再說。**

**Q3：要花多久才會覺得有用？**
整理第一份教案，當天就有感。但真正的威力要累積一陣子才看得到——所以重點是「開始，並且持續存進去」。

**Q4：那 Codex 呢？我也聽過。**
Codex 是另一套類似的工具，比較偏寫程式。這堂課先專心用 Antigravity 就好，不用同時學兩套，避免一開始就被淹沒。
